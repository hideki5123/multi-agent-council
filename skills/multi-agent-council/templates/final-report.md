## Template Variables Reference

| Variable | Type | Description |
|----------|------|-------------|
| `run_id` | string | Unique identifier for this council run |
| `timestamp` | string | ISO 8601 timestamp |
| `mode` | string | "architecture" or "review" |
| `providers_list` | string | Comma-separated list of providers |
| `final_summary` | string | 2-3 sentence executive summary |
| `decision_outcome` | string | "adopt", "reject", "need-info" (arch) or findings (review) |
| `recommendation` | string | Main recommendation text |
| `rationale_summary` | string | Why this conclusion was reached |
| `alternatives` | array | Alternative options considered |
| `risks` | array | Synthesized risks with severity |
| `disagreements` | array | Key model disagreements |
| `uncertainties` | array | Areas of low confidence |
| `next_actions` | array | Recommended follow-up actions |
| `rounds_completed` | number | How many rounds were executed |
| `total_duration_ms` | number | Total execution time |
| `short_circuit` | object | Short-circuit info if triggered |
